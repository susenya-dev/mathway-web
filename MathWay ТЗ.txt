Проект: MathWay Web - тренажёр ЕГЭ по математике

1. Цель

Создать веб-приложение для подготовки к тестовой части ЕГЭ по математике с автопроверкой, статистикой и генерацией вариантов.

2. Техническая основа проекта

Язык программирования backend: Python 3
Frontend: HTML, CSS, Bootstrap
ORM: SQLAlchemy
Хранение данных: .sqlite
Основные библиотеки: Flsak

3. Возможности веб-приложения

Прохождение тестовой части ЕГЭ по математике
Создание индивидуальных вариантов
Отслеживать прогресс
Получать автоматическую оценку решений 

4. Этапы разработки

Создание основы сайта 
Создание HTML-шаблонов страниц (Bootstrap)
Настройка БД и ORM (SQLAlchemy, модели)
Реализация регистрации и авторизации
Наполнение БД заданиями ЕГЭ 
Создание сервера с заданиями ЕГЭ
Создание REST API
Реализация проверки ответов и сохранения результатов
Реализация генерации случайных вариантов
Реализация статистики пользователя 
Тестирование и выгрузка на хостинг
