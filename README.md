MathWay Web

Сайт-тренажер для подготовки к ЕГЭ по математике, позволяющее пользователям:

- проходить задания ЕГЭ из тестовой части
- формировать индивидуальные варианты
- отслеживать прогресс
- получать автоматическую проверку ответов

На сайте реализованы: система авторизации, работа с базой данных, API и вебинтерфейс.
